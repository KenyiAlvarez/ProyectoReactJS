1. Clonar el repositorio

### `npm install`

3. Para ejecutar el proyecto utilice:

### `npm start`

O estara disponible en: [http://localhost:3000](http://localhost:3000)

 - Uso de FireBase para los datos y guardar lar ordenes que genera el usuario
